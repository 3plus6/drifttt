# Contributing

drifttt is maintained by a small individual project.

The first goal of this repository is to collect feedback in one place.

## How to contribute

- Use Issues for bug reports, feature ideas, and usage feedback.
- Please include your macOS version, drifttt version, and the app where the issue happened when reporting a bug.
- Pull requests are welcome, but large changes may not be merged immediately.

## Development policy

The maintainer decides whether and when to implement changes.

Features that affect privacy, input behavior, API cost, or macOS permissions will be handled carefully and may take time.

## Commercial redistribution

The source is public for review, learning, and improvement. Selling drifttt itself, a paid redistribution, or a substantially identical app is not allowed. See `LICENSE.md`.
