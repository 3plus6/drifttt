# Contributing / コントリビュート

## 日本語

driftttは、小さな個人プロジェクトとして開発しています。

まずは、利用者からの不具合報告、改善提案、感想をGitHub Issuesに集約することを目的としています。

### 参加方法

- 不具合報告、改善提案、感想はIssuesに投稿してください。
- 不具合報告では、macOSのバージョン、driftttのバージョン、入力先アプリ、再現手順を書いてください。
- Pull Requestも歓迎します。ただし、大きな変更はすぐに取り込めない場合があります。

### 開発方針

変更を取り込むかどうか、いつ取り込むかはメンテナーが判断します。

プライバシー、入力挙動、API料金、macOS権限に関わる変更は慎重に扱います。

### 商用再配布について

ソースコードは、内容確認、学習、個人利用、改善提案のために公開しています。drifttt自体、有料再配布版、または実質的に同じアプリを販売・有償提供することは許可していません。詳しくは `LICENSE.md` を確認してください。

## English

drifttt is maintained as a small individual project.

The first goal of this repository is to collect bug reports, feature ideas, and feedback through GitHub Issues.

### How to contribute

- Use Issues for bug reports, feature ideas, and usage feedback.
- For bug reports, include your macOS version, drifttt version, target app, and reproduction steps.
- Pull requests are welcome, but large changes may not be merged immediately.

### Development policy

The maintainer decides whether and when to implement or merge changes.

Changes that affect privacy, input behavior, API cost, or macOS permissions will be handled carefully.

### Commercial redistribution

The source is public for review, learning, personal use, and improvement. Selling drifttt itself, a paid redistribution, or a substantially identical app is not allowed. See `LICENSE.md`.
